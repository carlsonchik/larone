# A2A + LAR-1 message example

Minimal example of LAR-1 as an A2A typed data part.

```bash
cd demos/a2a-lar1
npm install
npm start
```

Uses [`@lar-1/a2a`](../../packages/lar1-a2a/) — no live A2A server required; prints wire-format payloads for copy into A2A clients.
